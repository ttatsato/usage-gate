# Report a security issue / セキュリティ報告
The project team welcomes security reports and is committed to providing prompt attention to security issues
本プロジェクトではセキュリティに関する報告とコミットは大歓迎です

# Security advisories / セキュリティ情報
Remediation of security vulnerabilities is prioritized by the project team. The project team endeavors to coordinate remediation with third-party stakeholders, and is committed to transparency in the disclosure process. 

プロジェクトチームは、セキュリティ上の脆弱性の修正を最優先事項としています。  
プロジェクトチームは、サードパーティのステークホルダーと連携して修正に取り組むとともに、開示プロセスにおける透明性を確保することに努めています。
